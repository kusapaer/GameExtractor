Replace Files

Back to Index

Replaces files in an archive with files on your computer.

A list of files and directories will be displayed in the Side Panel. Choose one or more files in the Side Panel, then click either the Replace Selected or Replace Matching button, depending on the type of replace that you want to perform. Note that the filenames won't change, only the content of the files.

Replace Selected: Select one or more files in the file list on the left, that you want to replace with a file on your computer. Select the new file in the list on the right. Clicking the Replace Selected button will replace all the selected files on the left with the selected file on the right.
Replace Matching: Select a number of files and directories in the list on the right, and click the Replace Matching button. If a file in your selection has the same name as a file in the archive, the file in the archive will be replaced with the matching file on the computer.

You can also replace files by selecting and dragging files from the Side Panel into the File List, then choosing one of the Replace Files options.
