Image Investigator

Back to Index

Shows a Image Investigator representation of the selected file in the File List.

The Image Investigator allow you to display an unknown file as an image. This is useful for files that contain image data somewhere within them, but where the file itself isn't a standard or known file format. You can use the fields to describe the offset, width, and height of the image data in the file, as well as the format of the image data.


This is the button to show a file in the Image Investigator
