Search or Filter

Back to Index

Searches for files in the archive. This can be done by either selecting the files that match, or by filtering the File List to only show you the matching files.

In the input field at the top, write the value that you are searching for. Under that, tick the fields that you want to search in.


This is the button to open the Search or Filter

There are 3 types of actions that you can perform:

Find Next Match: Finds the next matching file in the archive, and selects it.
Select All Matches: Selects all the files that match the search text.
Filter Files: The File List is filtered, so that it only shows the files that match the search text.
