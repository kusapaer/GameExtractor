Analyze Directory

Back to Index

Analyzes the files in a directory, looking for archives and previewable file types. This is most useful to someone wanting to get an overview of a whole game at once, or someone who wants to generate a list of all files in a game.

There are several checkboxes that allow you to choose how detailed you want the analysis:

Analyze Sub-Directories: When ticked, the current directory will be analyzed, as well as all sub-directories within it. When unticked, only the current directory is analyzed.
Identify Previewable Formats: When ticked, the analysis looks for archives as well as other files that can be previewed by Game Extractor, such as images and audio files. When unticked, only archives are identified.
Identify Formats within Archives: When ticked, all the files in an archive are analyzed, looking for any that can be previewed by Game Extractor. When unticked, the files in an archive are listed, but not whether they can be previewed or not.
Extract all Files from Archives: When ticked, the files are extracted from all archives that are found. Files are also converted into a different format, if chosen. When unticked, files are only extracted if required for some of the other analysis options chosen.

At the bottom, you can optionally choose a format for image and model conversions. If you have the Extract All Files checkbox ticked, all images and models that are found in archives will also be converted into these formats.

You also choose a format for the generated analysis report, and enter a report filename. The report provides a list of all identified archives, and other information depending on the checkboxes that were chosen.

The analysis report, as well as all files extracted (and optionally converted) are saved to the Analysis directory specified in the Options. To conserve disk space, if there are files in this directory when you close Game Extractor, you will be asked whether Game Extractor should remove them or not.
